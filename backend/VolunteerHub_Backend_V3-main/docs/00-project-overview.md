# Volunteer Hub - Project Overview

## Purpose
Volunteer Hub is a platform connecting volunteers, organizers, sponsors, and admins in a transparent volunteer event ecosystem.

## Main actors
- Volunteer
- Organizer
- Sponsor
- Admin

## Core modules
- Auth
- Volunteer Profile
- Organizer Management
- Event Management
- Application Approval
- Attendance
- Certificate and Badge
- Sponsor
- Rating and Feedback
- Notification
- Admin

## Tech stack
- ASP.NET Core MVC
- EF Core
- My SQL
- ASP.NET Core Identity

## Architecture rules
- MVC in Web layer
- Business logic in Application layer
- Entities in Domain layer
- Persistence and external services in Infrastructure layer

## Non-goals
- Do not build mobile app code
- Do not build external frontend SPA in this repo
