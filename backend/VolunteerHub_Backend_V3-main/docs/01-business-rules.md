# Business Rules

1. A volunteer can apply only once to the same event.
2. Only verified organizers can publish events.
3. Event start time must be earlier than end time.
4. Volunteer hours are counted only after approved attendance.
5. Certificate can be generated only after event completion and approved participation.
6. Rating is two-way between volunteer and organizer.
7. Admin can override moderation states when necessary.
8. Sponsor can track donations linked to specific events or campaigns.
